# AmixCLI Tool
- Allows you to use Amix in Command Line Interface since Amix is a Library.

# Installing
- To install it, it's simple, just run:
  ```
  ./installer.sh
  ```
  * it'll install the amix in $PREFIX/bin

# Using
- To use it, it's also simple, just run:  
  (the vertical-root argument is optional)  
  ```bash
  amix screen.amx --vertical-root
  ```
